# ZBrain charter

ZBrain is a local-first brain/RAG tool for markdown work memory. M0 creates the measurement harness and local-only privacy smoke test before retrieval implementation grows.

Primary user: a developer/leader with local docs who needs reliable retrieval with explicit provenance and no surprise egress.

Top workflows:

1. Find exact prior session/plan/decision.
2. Retrieve project status and supporting docs.
3. Demonstrate local-only smoke behavior to reviewers.

Switch trigger: ZBrain beats QMD BM25 on the frozen benchmark and has a credible path toward gbrain-quality retrieval without gbrain-level complexity.
